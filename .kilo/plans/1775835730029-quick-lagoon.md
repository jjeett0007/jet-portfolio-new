# Next Steps & Customization Guide

## Quick Start

Your template is a Next.js resume/portfolio site. Here's how to customize it:

### 1. Personal Information (Priority: High)
Edit `src/data/profile.ts`:
- Name, title, email, location
- Photo (place in `public/images/profile.jpg`)
- Social links (GitHub, LinkedIn, Twitter, etc.)
- Professional summary & highlights

### 2. Core Customization Files

| File | Purpose |
|------|---------|
| `src/data/profile.ts` | Personal info, photo, summary |
| `src/data/experience.ts` | Work history |
| `src/data/skills.ts` | Skills list |
| `src/data/education.ts` | Education/certifications |
| `src/data/projects.ts` | Portfolio projects |
| `src/config/site.config.ts` | Theme, features, navigation |

### 3. Theme Customization

In `src/config/site.config.ts`, modify:

```typescript
theme: {
  primaryColor: '220 92% 50%',  // HSL format - change the hue
  darkModeEnabled: true,
  fontFamily: 'Inter',  // Inter, Roboto, Open Sans, Poppins, Montserrat
}
```

### 4. Feature Toggles

In `site.config.ts`, enable/disable sections:
- `portfolio` - Show portfolio projects
- `contactForm` - Contact form
- `pdfDownload` / `printButton` - Export options
- `skillBars` - Visual skill bars vs list
- `certifications` / `languages` / `awards` - Additional sections
- `sideNav` - Desktop side navigation

### 5. Development Commands

```bash
bun install    # Install dependencies
bun build      # Build for production
bun lint       # Check code quality
bun typecheck # Type checking
```
